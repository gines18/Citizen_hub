## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/Citizen_hub.git
    ```
2. Navigate to the project directory:
    ```bash
    cd Citizen-hub
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
